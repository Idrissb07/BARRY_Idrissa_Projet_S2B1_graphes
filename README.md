# Graph Analysis Tool

This project was developed as part of a university course on graph theory.

The goal of this project is to create, visualize and analyze graphs using Python
and the NetworkX library.

## Features
- Graph visualization
- Degree and average degree calculation
- Diameter calculation
- Regular graph detection
- Graph creation from adjacency matrix
- Graph creation from adjacency list

## Technologies
- Python
- NetworkX
- Matplotlib
- NumPy

## How to run
pip install -r requirements.txt  
python main.py
